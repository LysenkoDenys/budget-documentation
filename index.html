Budget.app

General purpose

Budget is a modern, lightweight Progressive Web App (PWA) designed to help individuals track, manage, and analyze their financial activities with ease. It provides a clean, intuitive interface for recording transactions, categorizing expenses, setting up future financial goals, and visualizing spending patterns over time.

The application emphasizes speed, accessibility, and offline availability thanks to its PWA capabilities. Built with Next.js and React.js, it delivers a smooth user experience while ensuring scalability for future enhancements.

Getting started

Make sure you have the following installed before proceeding:

  • Node.js (v18 or later recommended)
  • npm (comes bundled with Node.js)
  • Yarn (Optional) if you prefer it over npm

You can check if Node.js and npm are installed by running:

node -v
npm -v

If not installed, you can download and install Node.js from nodejs.org.

To run the project on your PC you must have installed Node.js also you should write in the VSC terminal following:

Install dependencies:

npm install

Run the app in development mode

npm run dev

Build for production:

npm run build
npm start

Testing

npm test
